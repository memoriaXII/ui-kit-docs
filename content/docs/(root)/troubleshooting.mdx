## Components render unstyled

Make sure you have imported the kit's base stylesheet at your app entry:

```ts
import "@freedom/ui-kit/styles.css"
```

If you are also using a packaged theme, import its CSS too:

```ts
import "@freedom/ui-kit/themes/freedom.css"
```

## Less files fail to compile

The kit ships less sources. Configure your bundler to handle `.less`:

- Vite: add `vite-plugin-less` or `@arco-design/vite-plugin-less` (already in our examples)
- Next.js: use `next-with-less` or transpile via `next-transpile-modules`

## arco-design components are missing

`@arco-design/mobile-react` and `@arco-design/mobile-utils` are peer dependencies. Install them yourself in your app:

```bash
pnpm add @arco-design/mobile-react @arco-design/mobile-utils
```
