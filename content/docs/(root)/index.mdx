Freedom UI Kit is a themeable, host-agnostic React UI kit built on top of [Arco Design Mobile](https://arco.design/mobile/react) for Boxo and partner mini-apps.

Install via your favourite package manager:

```bash
pnpm add @freedom/ui-kit
```
