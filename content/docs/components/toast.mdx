## Usage

`Toast` is **not a React component**. It exposes two static methods that mount a transient overlay.

```tsx
import { Toast, PrimaryButton } from "@appboxo/ui-kit"

export function Example() {
  return (
    <PrimaryButton
      text="Save"
      onClick={() => Toast.info("Saved")}
    />
  )
}
```

## Methods

| | Visual | Use for |
|---|---|---|
| `Toast.info(msg)` | Neutral pill | Confirmations, success messages |
| `Toast.error(msg)` | Tinted danger pill | Failures, validation errors |

Both auto-dismiss after 2000 ms and anchor to the top of the viewport. Content is rendered via the kit's `Footnote2` typography.

## Theme tokens

Required: `--toast-background`, `--toast-text-color`, `--toast-*`.
Optional: `--boxo-toast-error-*` (falls back to `--toast-*`).
