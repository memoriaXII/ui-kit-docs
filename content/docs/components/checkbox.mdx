## Usage

Forwards every Arco Checkbox prop and adds an `iconType` for the kit's custom marks.

```tsx
import { Checkbox } from "@appboxo/ui-kit"

export function Example() {
  return (
    <Checkbox
      iconType="circle-dot"
      defaultChecked
      onChange={(checked) => console.log(checked)}
    >
      Accept terms
    </Checkbox>
  )
}
```

## `iconType`

- `"circle-dot"` — solid filled circle (default)
- `"circle-outline-dot"` — outlined ring
