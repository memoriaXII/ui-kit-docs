## Installation

```bash
pnpm add @appboxo/ui-kit
```

## Theme tokens

Required: `--boxo-card-mobile-background`, `--boxo-card-mobile-active-background`.
Optional: `--boxo-card-mobile-padding`, `--boxo-card-mobile-border-*`, `--boxo-card-mobile-shadow`.
