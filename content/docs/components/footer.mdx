## Installation

```bash
pnpm add @appboxo/ui-kit
```

Place inside `Layout`'s or `ResponsiveLayout`'s `footer` slot for the full sticky behaviour. Standalone use works too but the safe-area padding assumes a full-screen container.

## Theme tokens

`--footer-padding`, `--footer-gap`, `--footer-scrollable-shadow` (all required).
