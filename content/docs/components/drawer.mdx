---
title: Drawer
description: Bottom sheet for confirmations, pickers and contextual flows. Wraps Arco's PopupSwiper.
---

<ComponentPreview name="drawer-demo" description="Drawer trigger." />

## Usage

```tsx
import { Drawer, Flex, PrimaryButton, SecondaryButton } from "@appboxo/ui-kit"

export function Example() {
  const [open, setOpen] = useState(false)

  return (
    <>
      <PrimaryButton text="Confirm purchase" onClick={() => setOpen(true)} />
      <Drawer close={() => setOpen(false)} visible={open}>
        <Flex vertical gap={16} className="p-6">
          <div>Redeem 1,200 points for a $12 voucher?</div>
          <Flex gap={8}>
            <PrimaryButton text="Confirm" onClick={() => setOpen(false)} />
            <SecondaryButton text="Cancel" onClick={() => setOpen(false)} />
          </Flex>
        </Flex>
      </Drawer>
    </>
  )
}
```

## `hideControl`

Hide the drag handle at the top of the sheet:

```tsx
<Drawer hideControl visible={open} close={() => setOpen(false)}>
  {/* ... */}
</Drawer>
```

Swipe-down to dismiss is wired automatically.
