## Usage

```tsx
import { DatePicker } from "@appboxo/ui-kit"

export function Example() {
  const [value, setValue] = useState<number | null>(null)
  const today = Date.now()
  const inOneYear = today + 365 * 24 * 60 * 60 * 1000

  return (
    <DatePicker
      value={value}
      minTs={today}
      maxTs={inOneYear}
      onChange={(ts) => setValue(ts)}
    />
  )
}
```

`value` is a unix ms timestamp. Labels are formatted via `date-fns` (`dd MMM yyyy`).

## Requires UIKitProvider

The picker reads `useUIKitTranslation()` for the OK / cancel labels. Wrap the tree in `<NextUIKitProvider>` (Next) or `<UIKitProvider translation={useTranslation}>` (Vite / CRA) so translations resolve correctly.
