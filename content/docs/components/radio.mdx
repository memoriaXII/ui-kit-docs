## Usage

`Radio` is intentionally just the **visual mark** (a circle that fills when active). It does **not** ship a `RadioGroup` — wire your own selection state and place a `Radio` inside each option.

```tsx
import { Radio, Flex, TouchCell } from "@appboxo/ui-kit"

const options = ["card", "paypal", "apple"]

export function Example() {
  const [selected, setSelected] = useState("card")

  return (
    <Flex vertical gap={8}>
      {options.map((value) => (
        <TouchCell key={value} onClick={() => setSelected(value)}>
          <Flex align="center" gap={8}>
            <Radio active={selected === value} />
            <span>{value}</span>
          </Flex>
        </TouchCell>
      ))}
    </Flex>
  )
}
```
