## Default

Picks mobile vs desktop layers automatically via the kit's `useBreakpoint` hook. Shown here in a phone frame; on a real desktop viewport the same component renders a centered max-width column.

<ComponentPreview name="responsive-layout-default" />

## Installation

```bash
pnpm add @appboxo/ui-kit
```

## Theme tokens

`--boxo-layout-mobile-padding`, `--boxo-layout-mobile-footer-*` (all required).
