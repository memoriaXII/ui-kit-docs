## Installation

```bash
pnpm add @freedom/ui-kit
```

## Usage

```tsx
import { Button } from "@freedom/ui-kit"

export function Example() {
  return <Button variant="primary">Confirm</Button>
}
```

## Variants

`Button` exposes the variants used across all Boxo partner mini-apps. Use `primary` for the dominant call to action on a screen and at most one `destructive` per view.
