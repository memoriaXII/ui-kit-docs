## Installation

```bash
pnpm add @appboxo/ui-kit@npm:@rex-taiwan/ui-kit
```

## Usage

Buttons are **separate exports per variant**. Pass copy via `text` (not children). All variants accept an optional `href` + `replace` that route through `<UIKitLink>` (Next's `<Link>` when wrapped in `<NextUIKitProvider>`, plain `<a>` otherwise).

```tsx
import {
  PrimaryButton,
  SecondaryButton,
  TertiaryButton,
  QuaternaryButton,
} from "@appboxo/ui-kit"

export function Example() {
  return (
    <>
      <PrimaryButton text="Continue" onClick={() => {}} />
      <SecondaryButton text="Skip" />
      <TertiaryButton text="More" />
      <QuaternaryButton text="Filter" />

      {/* As a link */}
      <PrimaryButton text="Open" href="/checkout" />
    </>
  )
}
```

## Variants

| Component | Use for |
|---|---|
| `PrimaryButton` | The single dominant call-to-action on a screen |
| `SecondaryButton` | Supporting action sitting beside primary (e.g. Skip / Cancel) |
| `TertiaryButton` | Inline action inside a card or list row |
| `QuaternaryButton` | Outline filter / chip-like action |

## Theme tokens

`--button-primary-*`, `--button-default-*`, `--button-ghost-*` (Arco-managed) plus `--tertiary-button-*` and `--quaternary-button-*` (kit-only).
