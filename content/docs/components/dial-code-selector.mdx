---
title: DialCodeSelector
description: Controlled bottom-sheet popup that lets the user pick a country dialing code.
---

## Default

Controlled popup — you own the `visible` boolean and listen for `onSelect`. Country list comes from `libphonenumber-js` by default; pass `countriesData` for a localized list.

<ComponentPreview name="dial-code-selector" />

## Installation

```bash
pnpm add @appboxo/ui-kit
```

## Theme tokens

`--boxo-dial-code-selector-item-active-background` (required).
