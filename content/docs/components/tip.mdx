## Usage

```tsx
import { Tip } from "@appboxo/ui-kit"

export function Example() {
  return (
    <Tip text="We only use this on your device. Nothing leaves the phone." />
  )
}
```

## Emphasised prefix

```tsx
<Tip
  emphasisPrefix="Heads up:"
  text="You can stack rewards from multiple partners on the same Boxo Pass."
/>
```

## Custom icon

Pass any SVG-as-React-component (the kit's icons are already such components):

```tsx
import { Tip, InfoCircleIcon } from "@appboxo/ui-kit"

<Tip icon={InfoCircleIcon} text="Read the fine print." />
```

The icon defaults to the kit's `InfoCircleIcon`.
