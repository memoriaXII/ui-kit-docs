## Default

Use when [Drawer](/docs/components/drawer)'s defaults aren't right (custom direction, full-screen popups, non-bottom sheets). You manage the inner surface — including its background, padding and border radius.

<ComponentPreview name="popup-swiper-default" />

## Installation

```bash
pnpm add @appboxo/ui-kit
```

For the standard bottom-sheet confirmation pattern, reach for [Drawer](/docs/components/drawer) first.
